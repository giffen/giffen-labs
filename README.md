# Giffen Labs

This is where everything else lives.

## What this is

Giffen Labs is a sandbox — no positioning pressure, no audience, no narrative to maintain.
It's the place where I build things to learn, experiment with stacks I'm curious about,
and ship projects that don't need to justify themselves strategically.

If it's weird, half-finished, or only useful to three people, it lives here.

## The split

**giffen.me** is my operator/CTPO signal. Every post and project there answers one
question: why hire this person to own product and tech at a Series A/B company in a
regulated industry. It's curated and intentional.

**giffenlabs.ca** (this) is everything else. Green Building Gamer, Brown Note,
drum tools, proximity apps, random scrapers — things I built because I was curious
or someone asked or I wanted to try a new stack. Not a portfolio. Just a record of
what I've been making.
